# homebrew-tap

This is a personal Homebrew tap for custom tools.

## Installation

To add this tap to Homebrew:

```bash
brew tap rotty3000/tap
```

## Available Formulas

### [durl](https://github.com/rotty3000/durl)

A simple command-line tool for parsing and formatting URLs, implemented in Rust.

```bash
brew install durl
```

### [verz](https://github.com/rotty3000/verz)

A simple command-line tool for managing semantic versions, implemented in Rust.

```bash
brew install verz
```
